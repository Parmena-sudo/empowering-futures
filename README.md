# Empowering Futures Full-Stack Website

Complete frontend + backend for **Grassroots Empowerment Network — Empowering Futures**.

## Stack
Frontend: React + Vite + Tailwind CSS  
Backend: Node.js + Express + MongoDB  
Forms: Contact, Volunteer/Partner, Donation Interest  
Email: Optional Nodemailer notifications to grassrootsempowermentnetwork@gmail.com

## Run Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

## Run Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

Frontend: http://localhost:5173  
Backend: http://localhost:5000/api/health
